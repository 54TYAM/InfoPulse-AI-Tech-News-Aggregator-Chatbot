📰 InfoPulse AI – Tech News Aggregator Chatbot

InfoPulse AI is an AI-powered Tech News Aggregator Chatbot built using Python that collects, summarizes, and delivers the latest technology news and trends in real time through an interactive conversational interface.

The chatbot helps users stay updated with AI, cybersecurity, startups, programming, gadgets, cloud computing, and other trending technology topics using intelligent news aggregation and AI-generated summaries.

🚀 Features
📰 Real-time technology news aggregation
🤖 AI-powered chatbot interaction
📌 Latest trending tech updates
⚡ Fast and intelligent news summarization
🔍 Category-based news filtering
🌐 Interactive conversational interface
📱 Lightweight and user-friendly application
🧠 Personalized tech news recommendations

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/your-username/InfoPulse-AI.git
2️⃣ Navigate to the Project Folder
cd InfoPulse-AI
3️⃣ Create a Virtual Environment
python -m venv venv
4️⃣ Activate the Virtual Environment
Windows
venv\Scripts\activate
Mac/Linux
source venv/bin/activate
5️⃣ Install Dependencies
pip install -r requirements.txt
6️⃣ Add API Keys
Create a .env file and add your API keys:
NEWS_API_KEY=your_news_api_key
GEMINI_API_KEY=your_gemini_api_key
7️⃣ Run the Application
python app.py

💡 How It Works
The chatbot fetches the latest technology news using News APIs.
AI processes and summarizes the news articles.
Users interact with the chatbot through natural conversations.
The chatbot delivers personalized and real-time tech updates.
